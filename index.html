    Welcome to my KPI-ML fork.

    KPI-ML is an XML implementation of the ISO 22400 standard for Automation systems integration - Key performance indicators (KPIs) for manufacturing operations management. This fork is where I work on enhancements and revisions to the MESA KPI-ML standard, and particularly on making software implementations easier in C# and Java. See the MESA GitHub site for official releases.

    Authors and Contributors

    I collaborate mainly with @Emers, @Dennis-Brandl, and @JoeDo on the MESA XML Committee, and with others on the MESA APM (Asset Performance Management) Committee.
